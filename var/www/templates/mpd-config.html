V.A.M.P. Playback Configuration

If you mess up with this configuration you can reset to default. After resetting, and selecting the correct settings, please press save!

Вы можете сбросить настройки по умолчанию. После сброса и выбора правильных настроек, пожалуйста нажмите сохранить!

Audio Output

Here you can select your desired output option. V.A.M.P. supports only one Output at the time.

Выберите ваш USB ЦАП, или USB конвертер. Вывод одновременно осуществляется только на одно устройство.


Volume control mixer

Default - disabled. However, if you cannot set the Volume, switching to Software Volume Control will solve your issue, costing in decreased quality.

Настройку следует использовать, есть необходимо изменять громкость программно, или ЦАП имеет аппаратную поддержку регулировки на уровне USB, что встречается очень редко.

Hardware - Works only with some DACS and enables volume control without affecting sound quality. Best of both worlds.
Software - Works with every DAC available, but affectes audio quality.
Disabled - No remote volume control available, preserves audio quality but harms couch potatoes.
General music daemon options
If you have a problem with your MP3s ending abruptly it is recommended that you set this argument to "no" to attempt to fix the problem. Попробуйте изменить значение параметра, если только имеете проблемы с воспроизведением окончания mp3 файлов.
Enable DSD over PCM. If your DAC supports Native DSD, and there is in this list, turn off this option. Включение DSD over PCM. Если ваш цап не поддерживает этот режим, или работаев в DSD NATIVE (подробнее здесь), то параметр нужно отключить.
Very high values can lead to non-stability of the system! Установите приоритет аудиовыхода. Один из дополнительных параметров для опции реального времени. Очень высокие значения могут привести к нестабильности системы!
If yes, V.A.M.P. will normalize the volume of songs as they play. The default is no. Если да, V.A.M.P. нормализует громкость песен во время игры. По умолчанию нет.
This specifies the size of the audio buffer in kibibytes. The value of 60000 is the maximum. Этот параметр определяет размер звукового буфера в кибибайтах. Значение 60000 является максимальным.
This specifies how much of the audio buffer should be filled before playing a song. Это указывает, какая часть звукового буфера должна быть заполнена перед воспроизведением песни.
This setting enables automatic update of music database when files in music_directory are changed. Этот параметр позволяет автоматически обновлять базу данных музыки при изменении файлов. Не рекомендуется, так как нагружает процессор.
Resampling

Here you can set Up or Downsampling. Set the desired bit-depth and resolution. By default, this setting is disabled. Not recommended for use.

Здесь вы можете установить частоту и разрешение ресемплинга. По умолчанию отключено.
